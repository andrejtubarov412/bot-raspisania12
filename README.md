# bot-raspisania12# 📋 Бот для управления расписанием сотрудников

Веб-приложение для автоматизации составления расписания и учёта рабочего времени сотрудников.  
Сотрудники оставляют пожелания по дням и сменам, администратор одним нажатием генерирует расписание, а также ведёт учёт отработанных часов.

---

## 🚀 Возможности

- **Для сотрудников:**
  - Регистрация через форму (достаточно указать имя).
  - Оставление пожеланий по дням недели и сменам (утро, день, ночь, вечер).
  - Просмотр итогового расписания.
  - Просмотр своих отработанных часов.

- **Для администратора:**
  - Автоматическая генерация расписания на основе пожеланий (с учётом заполнения всех смен).
  - Добавление отработанных часов вручную (с датой и сменой).
  - Очистка всех пожеланий.
  - Просмотр статистики, списка сотрудников, всех пожеланий и расписания.

bash

    Пароль по умолчанию: admin123

        Сгенерировать расписание

📌 Как пользоваться
Для сотрудников

    Откройте главную страницу.

    Нажмите «Оставить пожелание».

    Введите своё имя, выберите день недели и смену.

    Нажмите «Отправить».

    Чтобы посмотреть расписание, перейдите на страницу «Расписание».

    Чтобы увидеть свои часы, перейдите на страницу «Мои часы».

Для администратора

    Нажмите «Админ-панель» на главной.

    Введите пароль admin123.

    В админ-панели доступны:

        Сгенерировать расписание – автоматически распределит сотрудников по сменам.

        Очистить пожелания – удалит все текущие пожелания.

        Добавить часы – введите имя, дату, количество часов и смену.

        Просмотр всех сотрудников, пожеланий, расписания и часов.

🧩 Настройка параметров

Все настройки хранятся в файле config.py:

    SHIFT_HOURS – продолжительность каждой смены (часы).

    DAYS – список дней недели.

    SHIFTS – список смен.

    SCHEDULE_TEMPLATE – шаблон расписания: сколько человек нужно на каждую смену в каждый день.

Пример:
python

SCHEDULE_TEMPLATE = {
    'понедельник': {'утро': 1, 'день': 2, 'ночь': 1},
    'вторник': {'утро': 1, 'день': 2, 'ночь': 1},
    # ...
}

Измените эти параметры под свои нужды.
📊 База данных

    Используется SQLite, файл schedule.db создаётся автоматически.

    Таблицы:

        users – сотрудники.

        wishes – пожелания.

        schedule – готовое расписание.

        hours – отработанные часы.
